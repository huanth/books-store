# BookStore


## How to run project
`git clone https://github.com/huanth/books-store`

`cd books-store`

### Run frontend Angualar

`npm i`

`ng serve`

#### It run on `http://localhost:4200/`

### Run backend to demo data
`cd src/data`

`npx json-server db.json`

#### It run on `http://localhost:3000/`
